# Tamil-ChatBot
A simple tamil-chatbot using textblob translation.

## Packages used :
    chatterbot
    textblob
    pyqt5
    
## Useful Commands
    pip install chatterbot
    pip install -U textblob
    sudo apt-get install python-pyqt5 
To convert .ui file to .py use
    "pyuic5 -x INPUTFILENAME.ui -o OUTPUTFILENAME.py"
